載體

「旋舞炎劇團」2021 年度作品

以宇宙星體元素為概念，由元素碲-T（地球）硒-S（月亮）為主軸，結合 Wifi x Processing x Pixelpusher，將精心設計好的訊號源載入在機甲控制板內，可以從機甲胸口顯示動態影像，並以世界首創的「浮空環」(Hologram Hoop)、「浮空旗幟」(Hologram stick poi) 及動畫殘像 Virtual poi 呈現強烈的 Cyberpunk 風格視覺作品。
